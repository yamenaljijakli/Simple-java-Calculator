# 🧮 Simple Java Calculator
My first programming project in **Java**, built as part of my computer science journey at **AOU**.

---

### 🚀 Description
A basic console-based calculator that allows users to perform mathematical operations. This project helped me understand:
* **Java Basics:** Variables and Data Types.
* **Input/Output:** Using the `Scanner` class.
* **Logic:** Conditional `if-else` statements.

### 🛠️ How to Run
1. Make sure you have **JDK** installed on your machine.
2. Clone this repository or download the `Calculator.java` file.
3. Open your terminal or VS Code.
4. Compile the code:
   ```bash
   javac Calculator.java
